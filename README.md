# SuperHero Front-End Project

Projeto: Aplicação Front-End que consome a API superhero para exibir cards de personagens.
Tecnologias usadas: HTML, CSS, JavaScript e  API.


## Arquivos
- `index.html` — página principal.
- `styles.css` — estilos.
- `script.js` — lógica que consome a API e gera os cards dinamicamente.
- `README.md` — este arquivo.

## Como executar localmente
1. Extraia os arquivos (ou abra a pasta do projeto).
2. Recomendado: rode um servidor local para evitar problemas CORS:
   - Partir da pasta do projeto e abra `http://localhost:8000`
   - VSCode: Live Server
3. Abra `index.html` no navegador (via servidor local ou diretamente).
4. Use a busca para procurar personagens, ou veja os 20 personagens carregados inicialmente.

## Publicar no GitHub
1. Crie repositório no GitHub.
2. No terminal dentro da pasta do projeto:
   git init
   git add .
   git commit -m "Initial commit - SuperHero front-end project"
   git branch -M main
   git remote add origin https://github.com/<seu-usuario>/<seu-repo>.git
   git push -u origin main

3. Para publicar no GitHub Pages: Settings -> Pages -> escolha branch `main` / / (root) -> Salve. Aguarde alguns minutos.
